Requirements

Technical requirements include:

  • ExpressionEngine 2
  • Pixel & Tonic Matrix
  • PHP 5.2+
  • A modern browser: Firefox, Safari, Google Chrome or IE8+

Installation

  1. Download the latest version of NSM Photo Tags.
  2. Extract the .zip file to your desktop
  3. Copy system/expressionengine/third_party/nsm_photo_tags to system/expressionengine/third_party
  4. Copy themes/third_party/nsm_photo_tags to themes/third_party (you may need to create the third_party folder)
  5. Install and activate Pixel & Tonic Matrix
